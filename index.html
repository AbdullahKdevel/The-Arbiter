⚿ ANTHROPIC API KEY

Inside Claude.ai: leave empty — requests are handled for you.

On GitHub Pages / any static host: paste your key below. Get one at console.anthropic.com → API Keys. It lives only in this browser and is sent only to api.anthropic.com.

You play at a real table with real dice and books — The Arbiter is only the voice of your Dungeon Master. When a check is called, roll your own dice and tell it the number. As you play, the DM keeps the party sheets live — awarding XP, leveling characters, and recording new gear or spells — and pops a small notice each time a sheet changes.

Edition, tone, response length, and lines & veils belong to the current campaign; the key and model belong to this device. Long campaigns are quietly condensed into a running chronicle so the DM remembers old sessions without re-reading every word. Campaigns auto-save in this browser. Use ⇩ EXPORT / ⇧ IMPORT in the menu to carry a saga between devices. Deleting asks twice; exports never contain your key.
